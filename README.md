## ☕ About Java

Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems (now owned by Oracle). It follows the principle of **"Write Once, Run Anywhere (WORA)"**, meaning compiled Java code can run on all platforms that support the Java Virtual Machine (JVM).

Java is widely used for:

- Desktop Applications
- Web Applications
- Enterprise Software
- Android Development
- Banking & Financial Systems
- Multithreaded Applications
- Network Programming

### 🔹 Key Features of Java
- Object-Oriented
- Platform Independent
- Secure
- Robust & Reliable
- Multithreaded
- High Performance (with JIT compiler)
- Strong Memory Management (Garbage Collection)

Java programs are written in `.java` files, compiled into `.class` bytecode files, and executed using the Java Virtual Machine (JVM).
